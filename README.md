## 🐞 Bug Identified

### ❗ OTP Not Received (Pakistan Region)

**Description:**
While testing the signup process, the OTP was not received on the provided phone number when operating from Pakistan.

**Steps to Reproduce:**
1. Open signup page
2. Enter valid phone number
3. Request OTP
4. Wait for OTP

**Expected Result:**
OTP should be received within a few seconds.

**Actual Result:**
No OTP received.

**Severity:** High  
**Priority:** High
